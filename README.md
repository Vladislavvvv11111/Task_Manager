# Менеджер задач (To-Do List)

Консольное приложение на C++ для управления списком задач: добавление, просмотр,
фильтрация, сортировка, редактирование, удаление и отметка задач как выполненных.
Все задачи хранятся в текстовом файле и автоматически загружаются при запуске
программы. По умолчанию используется файл `tasks.txt`, но пользователь может
передать свой путь к файлу при запуске.

## Возможности программы

- Добавление задачи с описанием, дедлайном и уровнем важности.
- Просмотр всех задач в виде списка.
- Фильтрация задач по ключевому слову в описании.
- Фильтрация задач по уровню важности (1 — низкая, 2 — средняя, 3 — высокая).
- Фильтрация задач по статусу (выполнена / не выполнена).
- Сортировка задач по дедлайну (от самой ранней к самой поздней).
- Отметка задачи как выполненной.
- Изменение описания, дедлайна и важности задачи.
- Удаление задачи.

## Требования

- Компилятор с поддержкой C++17 (например, g++ или clang++).
- CMake версии 3.10 или новее.

## Сборка проекта

```bash
mkdir build
cd build
cmake ..
cmake --build .
```

После сборки в каталоге `build/` появятся два исполняемых файла:

- `task_manager` — основная программа;
- `run_tests` — программа с тестами.

## Запуск программы

```bash
./build/task_manager
```

Чтобы указать собственный файл для хранения задач, передайте путь первым
аргументом командной строки:

```bash
./build/task_manager data/my_tasks.txt
```

При первом сохранении программа создаст файл задач. При следующих запусках задачи
будут загружены из указанного файла.

## Запуск тестов

```bash
cd build
ctest --output-on-failure
```

## Пример работы программы

После запуска программы появляется меню:

```
===== МЕНЕДЖЕР ЗАДАЧ =====
1. Показать все задачи
2. Добавить задачу
3. Удалить задачу
4. Изменить задачу
5. Отметить задачу как выполненную
6. Фильтр по ключевому слову
7. Фильтр по важности
8. Фильтр по статусу
9. Сортировать по дедлайну
0. Выход
Выберите пункт меню:
```

### Добавление задачи

Выберите пункт `2`, затем введите описание, дедлайн в формате `ГГГГ-ММ-ДД` и
уровень важности от 1 до 3:

```
Выберите пункт меню: 2
Описание задачи: Сделать ДЗ по C++
Дедлайн (ГГГГ-ММ-ДД): 2026-06-20
Важность (1 - низкая, 2 - средняя, 3 - высокая): 3
Задача добавлена.
```

### Просмотр задач

```
Выберите пункт меню: 1
ID: 1 | Описание: Сделать ДЗ по C++ | Дедлайн: 2026-06-20 | Важность: 3 | Статус: не выполнена
```

### Фильтрация по ключевому слову

```
Выберите пункт меню: 6
Введите ключевое слово: ДЗ
ID: 1 | Описание: Сделать ДЗ по C++ | Дедлайн: 2026-06-20 | Важность: 3 | Статус: не выполнена
```

### Отметка задачи как выполненной

```
Выберите пункт меню: 5
Введите ID задачи для отметки выполненной: 1
Задача отмечена как выполненная.
```

## Формат хранения задач

Каждая задача хранится в файле `tasks.txt` в отдельной строке в формате:

```
id;описание;дедлайн;важность;статус
```

Например:

```
1;Сделать ДЗ по C++;2026-06-20;3;0
```

где `статус` равен `1`, если задача выполнена, и `0` — если не выполнена.
