Менеджер задач (To-Do List)

Консольное приложение на C++ для управления списком задач: добавление, просмотр, фильтрация, сортировка, редактирование, удаление и отметка задач как выполненных. Все задачи хранятся в текстовом файле и автоматически загружаются при запуске программы. По умолчанию используется файл tasks.txt, но пользователь может передать свой путь к файлу при запуске.

Возможности программы

  • Добавление задачи с описанием, дедлайном и уровнем важности.
  • Просмотр всех задач в виде списка.
  • Фильтрация задач по ключевому слову в описании.
  • Фильтрация задач по уровню важности (1 — низкая, 2 — средняя, 3 — высокая).
  • Фильтрация задач по статусу (выполнена / не выполнена).
  • Сортировка задач по дедлайну (от самой ранней к самой поздней).
  • Отметка задачи как выполненной.
  • Изменение описания, дедлайна и важности задачи.
  • Удаление задачи.

Требования

  • Компилятор с поддержкой C++17 (например, g++ или clang++).
  • CMake версии 3.10 или новее.

Сборка проекта

mkdir build
cd build
cmake ..
cmake --build .

После сборки в каталоге build/ появятся два исполняемых файла:

  • task_manager — основная программа;
  • run_tests — программа с тестами.

Запуск программы

./build/task_manager

Чтобы указать собственный файл для хранения задач, передайте путь первым аргументом командной строки:

./build/task_manager data/my_tasks.txt

При первом сохранении программа создаст файл задач. При следующих запусках задачи будут загружены из указанного файла.

Запуск тестов

cd build
ctest --output-on-failure

Формат хранения задач

Каждая задача хранится в файле tasks.txt в отдельной строке в формате:

id;описание;дедлайн;важность;статус

Например:

1;Сделать ДЗ по C++;2026-06-20;3;0

где статус равен 1, если задача выполнена, и 0 — если не выполнена.
